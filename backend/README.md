# Globe-Trotter Backend Setup

## Prerequisites
- Node.js and npm installed
- MongoDB running locally or provide a connection string in `.env` as `MONGO_URI`

## Install dependencies
```
npm install
```

## Run the backend server
```
npm run dev
```

Server runs on port 5000 by default.

## API Endpoints
- `/` : Health check
- `/api/test` : Sample API route
